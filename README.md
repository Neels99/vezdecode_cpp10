# vezdecode_cpp10
Вездекод 10

# Install and launch
```
sudo apt update
sudo apt install git
sudo apt install qt5-default
sudo apt install libqt5webkit5-dev
sudo apt install build-essential
git clone https://github.com/Neels99/vezdecode_cpp10.git
cd vezdecode_cpp10
sudo mkdir build
cd build
qmake ..
make
./vezde10

```
